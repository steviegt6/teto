# teto

> **Kasane Teto** ([Japanese](https://en.wikipedia.org/wiki/Japanese_language): 重音テト, pronounced [\[kasane teꜜto\]](https://en.wikipedia.org/wiki/Help:IPA/Japanese)) is a virtual singer with a voicebank for [UTAU](https://en.wikipedia.org/wiki/UTAU), a singing voice synthesizer software.

---

**teto** is a set of projects/services mostly aimed at aiding in technical Terraria development (primarily oriented toward modding). Its primarily used through a Discord bot of the same name, with tools for debugging and programming as well as bringing up/looking up Terraria data.
